## Robot Youtube

## Description

This is a simple CRUD-like web API based on FastAPI, SQLALchemy and PostgreSQL.

## Usage

### Prerequisites
- Installed docker and docker-compose

### Run the app

```bash
git clone
docker-compose up
```

### Test the app

```bash
git clone
docker compose -f docker-compose.test.yml up --build
```
